# creat-qr-image
This is a node gadget for batch generation of QR codes, 
which can easily generate a large number of regular QR codes. 
快速批量生成二维码
## Install

```start
$ npm install
```
## Usage

You need a node environment. npm install.
set the basic link address and file directory.
1.use node.
$ node creatqr.js
2.use bat
click creat.bat
